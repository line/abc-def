## Installation

### 1. Install Tailwind CSS

ABC Def requires Tailwind CSS to be installed in your project.

```bash
npm install -D tailwindcss@^3
npx tailwindcss init
```

### 2. Install ABC Def Tailwind CSS Plugin

Install `@line/abc-def-tailwindcss` as a dev dependency:

```bash
npm install -D @line/abc-def-tailwindcss
# or
pnpm add -D @line/abc-def-tailwindcss
# or
yarn add -D @line/abc-def-tailwindcss
```

### 3. Configure Tailwind CSS

Add `@line/abc-def-tailwindcss` to your tailwind.config.js or tailwind.config.ts:

```js
// tailwind.config.js (CommonJS)
module.exports = {
  ...
  plugins: [require("@line/abc-def-tailwindcss")],
};
```

```ts
// tailwind.config.ts (ESM)
import type { Config } from "tailwindcss";

import abcDefTailwind from "@line/abc-def-tailwindcss";

export default {
  ...
  plugins: [abcDefTailwind],
} satisfies Config;
```
