## Installation

You need Node.js and Tailwind CSS installed to use this package.

1\. Install `@line/abc-def-tailwindcss` as a Node package:

```sh
npm install -D @line/abc-def-tailwindcss
or
yarn add -D @line/abc-def-tailwindcss
or
pnpm add -D @line/abc-def-tailwindcss
```

2\. Add `@line/abc-def-tailwindcss` to tailwind.config.js or tailwind.config.ts:

```js
// CommonJS
module.exports = {
  //...
  plugins: [require("@line/abc-def-tailwindcss")],
};
```

```js
// ESM
import abcDefTailwind from "@line/abc-def-tailwindcss";

export default {
  //...
  plugins: [abcDefTailwind],
};
```

If you want to use only HTML classes, please refer to [HTML Components](/docs/html_coming-soon--docs).  
If you are using a React or Vue framework, please refer to [React Components](/docs/react_component-list--docs) and [Vue Components](/docs/html_coming-soon--docs).
