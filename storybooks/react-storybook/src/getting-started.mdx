# Getting Started

## Prerequisites

ABC Def React requires **Tailwind CSS v4** and the ABC Def Tailwind CSS plugin. Follow the [Installation guide](/docs/getting-started-installation--docs) to set up Tailwind CSS v4 first.

## Installation

### 1. Install React Package

Install `@line/abc-def-react`:

```bash
npm install @line/abc-def-react
# or
pnpm add @line/abc-def-react
# or
yarn add @line/abc-def-react
```

### 2. Configure CSS Entry Point

In Tailwind CSS v4, you configure content paths directly in your CSS file using the `@source` directive.

Update your main CSS file (e.g., `src/app/globals.css` or `src/index.css`):

```css
@import "tailwindcss";
@plugin "@line/abc-def-tailwindcss";

/* Include ABC Def React components in content scanning */
@source "./src/**/*.{js,ts,jsx,tsx}";
@source "./node_modules/@line/abc-def-react/**/*.js";

/* Optional: Enable dark mode */
@custom-variant dark (&:is(.dark *));
```

> **Note:** Tailwind CSS v4 uses CSS-first configuration. You no longer need a `tailwind.config.js` or `tailwind.config.ts` file.

### 3. Framework-Specific Configuration

<details>
<summary>For Next.js 13 - 14:</summary>

Modify `transpilePackages` in your `next.config.js` file:

```js
const nextConfig = {
  // ... your content here
  transpilePackages: ["@line/abc-def-react"],
};

module.exports = nextConfig;
```

</details>

<details>
<summary>For Next.js 12:</summary>

Install next-transpile modules:

```bash
npm install next-transpile-modules
```

And import the package inside your `next.config.js` file:

```js
const withTM = require("next-transpile-modules")(["@line/abc-def-react"]);
```

Finally, you can wrap your module.exports using withTM like so:

```js
module.exports = withTM({
  //... your content here
});
```

</details>
