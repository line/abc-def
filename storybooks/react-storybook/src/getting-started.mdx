# Getting Started

1\. First, follow the `@line/abc-def-tailwindcss` [Installation guide](/docs/getting-started-installation--docs) to set up the configuration.

2\. Install `@line/abc-def-react` as a Node package:

```sh
npm install @line/abc-def-react
or
yarn add @line/abc-def-react
or
pnpm add @line/abc-def-react
```

3\. Check the [Components list](/docs/react_component-list--docs) to see all available components.
