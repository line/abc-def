# Getting Started

## Prerequisites

ABC Def React requires Tailwind CSS and the ABC Def Tailwind CSS plugin. Follow the [Installation guide](/docs/getting-started-installation--docs) to set up Tailwind CSS first.

## Installation

### 1. Install React Package

Install `@line/abc-def-react`:

```bash
npm install @line/abc-def-react
# or
pnpm add @line/abc-def-react
# or
yarn add @line/abc-def-react
```

### 2. Configure Tailwind CSS

Update your `tailwind.config.js` or `tailwind.config.ts` to include the ABC Def React package in the content array:

```js
// tailwind.config.js (CommonJS)
module.exports = {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@line/abc-def-react/**/*.js",
  ],
  plugins: [require("@line/abc-def-tailwindcss")],
};
```

```ts
// tailwind.config.ts (ESM)
import type { Config } from "tailwindcss";

import abcDefTailwind from "@line/abc-def-tailwindcss";

export default {
  content: [
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/@line/abc-def-react/**/*.js",
  ],
  plugins: [abcDefTailwind],
} satisfies Config;
```

### 3. Framework-Specific Configuration

<details>
<summary>For Next.js 13 - 14:</summary>

Modify `transpilePackages` in your `next.config.js` file:

```js
const nextConfig = {
  // ... your content here
  transpilePackages: ["@line/abc-def-react"],
};

module.exports = nextConfig;
```

</details>

<details>
<summary>For Next.js 12:</summary>

Install next-transpile modules:

```bash
npm install next-transpile-modules
```

And import the package inside your `next.config.js` file:

```js
const withTM = require("next-transpile-modules")(["@line/abc-def-react"]);
```

Finally, you can wrap your module.exports using withTM like so:

```js
module.exports = withTM({
  //... your content here
});
```

</details>
