# Security Policy

## Supported Versions

Security fixes are provided for the latest minor release of ABC Def 2.x.

## Reporting A Vulnerability

Please do not open a public issue for a suspected vulnerability.

Report security concerns through GitHub private vulnerability reporting when available, or contact the maintainers at <dl_oss_dev@linecorp.com>.

Include the affected package, version, reproduction steps, impact, and any known workaround. The maintainers will review the report and coordinate disclosure.
