# ABC Def 소개

## 프로젝트 개요

ABC Def (ABC Studio's Definitive Elements Foundation)는 효율적인 디자인 시스템입니다.

- 3-tier token 구조를 사용한 일관된 디자인
- Headless UI 기반의 유연한 커스터마이징
- Tailwind CSS 기반의 유틸리티 우선 접근

## 주요 특징

### Three-tier Token Structure

- Semantic tokens
- Component tokens
- Utility tokens

### Headless UI 기반

- 완전한 접근성 지원
- 유연한 스타일 커스터마이징
- 프레임워크 독립적 설계
